## Getting Started

First, run the development server:

```bash
npm run dev

```

# creat firebase account and enter firebase details in firebase.js file

firestore database , edit rules and add this line

```bash
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
```
